# بوت تليجرام NSFW AI Companion مثل HoneyChat

بوت تليجرام متقدم يشبه HoneyChat:
- شخصيات AI متعددة (قابلة للتوسعة)
- دردشة NSFW غير مقيدة (uncensored)
- مستويات محتوى (0-5) مثل HoneyChat
- ذاكرة محادثات (تاريخ + ملخص)
- توليد صور للشخصية باستخدام Pollinations.ai (يدعم NSFW)
- دعم اللغة العربية
- سهل التطوير والنشر

## المميزات في النسخة الحالية (v1)
- اختيار شخصية من قائمة جاهزة
- دردشة مستمرة مع ذاكرة (آخر 15 رسالة + ملخص)
- أمر /level لتغيير مستوى الصراحة (0 = آمن، 5 = hardcore صريح)
- أمر /image لتوليد صورة للشخصية حسب السياق
- /reset لمسح الذاكرة
- نظام prompt قوي للـ NSFW بدون رفض

## التقنيات المستخدمة
- Python + pyTelegramBotAPI (telebot)
- Groq API (LLM سريع ورخيص - Llama 3.1 أو Mixtral)
- Pollinations.ai لتوليد الصور (مجاني و uncensored)
- SQLite للحفظ الدائم (مستخدمين + محادثات + إعدادات)

## كيفية التشغيل محلياً

1. ثبت المتطلبات:
```bash
pip install -r requirements.txt
```

2. انسخ `.env.example` إلى `.env` وضع مفاتيحك:
```
TELEGRAM_BOT_TOKEN=your_bot_token_from_BotFather
GROQ_API_KEY=your_groq_key
```

3. شغل البوت:
```bash
python bot.py
```

## النشر على Railway / Render / VPS
- استخدم webhook (موصى به) بدلاً من polling لتوفير الموارد.
- في Railway: أضف المتغيرات البيئية، واستخدم `gunicorn` أو `uvicorn` إذا غيرت إلى FastAPI.
- إذا واجهت خطأ 409 Conflict: تأكد أنك لا تشغل أكثر من instance واحد، أو استخدم webhook مع secret token.

## خطوات التطوير القادمة (يمكنك طلبها)
- إضافة صوت (TTS) باستخدام ElevenLabs أو gTTS
- السماح للمستخدم بإنشاء شخصيات خاصة به
- تحسين الذاكرة باستخدام vector store أو تلخيص أفضل
- نظام اشتراكات / حدود مجانية
- دعم فيديو قصير (صعب لكنه ممكن عبر API)
- واجهة Mini App في تليجرام

## تحذيرات قانونية وأخلاقية
- هذا البوت للبالغين فقط (18+).
- لا تستخدمه لمحتوى غير قانوني (قاصرين، عنف حقيقي، إلخ).
- في بعض الدول قد تكون هناك قوانين حول bots الـ NSFW.
- تأكد من إضافة تحذير في /start.
